# jugg
A blog template

```bash
git clone https://github.com/jugg-yy/jugg.git
cd jugg
npm install

node bin/jugg.js create myBlog
node bin/jugg.js build myBlog
node bin/jugg.js server myBlog
```

访问 http://localhost:8001/ 预览
